# Recharge Plan
## Overview
This project is a simple Recharge Plan Website built using only HTML and CSS. It displays different mobile recharge plans in a clean and responsive layout.
## Features
* Display of multiple recharge plans
* Pricing cards with plan details
* Attractive UI using CSS styling
* Responsive design for mobile and desktop
* Clean and simple layout
## Technologies Used
* HTML5
* CSS3
## Project Structure
**act4_recharge_plan-main**
* index.html
* style.css
* README.md
* LICENSE

1. Download or clone the repository: git clone https://github.com/AL-FAIZKHAN007/act4_recharge_plan.git
